### Beschreibung ###
Dieses Modul verschickt eine Email an eine zuvor konfigurierte Email-Adresse wenn ein Fehler (Exception) im Shop
aufgetreten ist.

### Installation ###
Kopieren Sie alle Dateien aus dem Ordner "copy_this" in Ihr Shopverzeichnis.
Melden Sie sich im Admin an und navigieren zu Erweiterungen -> Module. Wählen Sie das Modul
aus und klicken Sie auf aktivieren.

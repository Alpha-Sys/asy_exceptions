### Beschreibung ###
Dieses Modul versendet eine Email an eine zuvor konfiguririerte Email-Adresse so bald eine Exception im Shop aufgetreten ist.
Somit muss das Exception-Log nicht manuell geprüft werden, sondern man wird automatisch informiert. 

### Installation ###
Kopieren Sie alle Dateien aus dem Ordner "copy_this" in Ihr Shopverzeichnis.
Melden Sie sich im Admin an und navigieren zu Erweiterungen -> Module. Wählen Sie den neuen Modul-Eintrag
aus und klicken Sie auf aktivieren.
